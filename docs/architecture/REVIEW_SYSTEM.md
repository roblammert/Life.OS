# REVIEW_SYSTEM.md  
**Review System Specification — Life.OS**  
*Part of the Life.OS Architecture Documentation Suite*

The Review System generates daily, weekly, monthly, and yearly reflections that summarize the user’s emotional, cognitive, behavioral, and productivity patterns. These reviews are powered by Life.Coach and integrate data from all modules: Journal, Notes, Tasks, Storage, Timeline, and the Life Graph.

This document defines the review generation pipelines, data sources, insight types, UI surfaces, and scheduling logic.

---

# 1. Purpose and Responsibilities

The Review System helps users understand their life across different time scales.

### Core Responsibilities
- Generate daily, weekly, monthly, and yearly reviews.
- Summarize emotional, cognitive, behavioral, and productivity patterns.
- Identify themes, trends, and anomalies.
- Highlight Life Moments and major events.
- Provide actionable insights and reflection prompts.
- Integrate with the Life Timeline and Life Graph.
- Store reviews locally and display them in the UI.

### Why Reviews Matter
Reviews help users:
- See long‑term patterns  
- Understand emotional cycles  
- Track habits and routines  
- Reflect on progress  
- Identify areas for improvement  
- Celebrate growth

# 3. Review Data Model

Reviews are stored locally in IndexedDB.

### `reviews`
| Field | Type | Description |
|-------|------|-------------|
| id | string | UUID |
| review_type | string | daily, weekly, monthly, yearly |
| period_start | number | Timestamp |
| period_end | number | Timestamp |
| summary | string | High‑level summary |
| insights | object[] | List of insights |
| moments | string[] | Life Moment IDs |
| stats | object | Numeric and categorical stats |
| prompts | string[] | Reflection prompts |
| created_at | number | Timestamp |

---

# 4. Review Generation Pipeline

Reviews are generated by Life.Coach using a multi‑stage pipeline.

## Stage 1 — Data Collection
Collect data from:
- Journal entries  
- Notes  
- Tasks  
- Workbooks  
- Timeline events  
- Life Graph nodes/edges  
- Coach insights  

## Stage 2 — Preprocessing
- Normalize text  
- Chunk long entries  
- Extract metadata  
- Aggregate stats  

## Stage 3 — Embedding & Clustering
- Generate embeddings  
- Cluster concepts  
- Identify themes  
- Detect semantic shifts  

## Stage 4 — Pattern Detection
- Emotional trends  
- Cognitive patterns  
- Habit consistency  
- Productivity cycles  
- Correlations  

## Stage 5 — Insight Generation
- Summaries  
- Highlights  
- Life Moments  
- Suggested tasks  
- Suggested routines  
- Reflection prompts  

## Stage 6 — Review Assembly
- Combine insights into a narrative  
- Format for UI  
- Store in IndexedDB

# 6. Review Scheduling

Reviews are scheduled using the Notification Engine.

### Daily Review
- Generated each morning  
- Delivered via notification  

### Weekly Review
- Generated at end of week  
- Delivered Sunday evening (default)  

### Monthly Review
- Generated on the 1st of each month  

### Yearly Review
- Generated on January 1st  

### User Controls
- Enable/disable each review type  
- Adjust delivery time  
- Snooze or postpone
